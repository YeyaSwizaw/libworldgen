Detailed Description

template<typename _t>
class WorldBase< _t >

The base class for all world types (World and EnhancedWorld).

This class should not be used directly - the classes that inherit from it should be used instead.

Template Parameters
_tThe world type that inherits this class, so the functions know what type to return for method chaining.
